MonoBehaviour

th vng Lifecycle ca MonoBehaviour

Tt c cc thnh phn c tha k t lp MonoBehaviour thc hin mt lot cc hm s kin theo mt th t xc nh. Cc hm ny bao gm:

Awake(): c gi sau khi mt i tng tr chi c khi to.
OnEnable(): c gi khi mt i tng tr chi c kch hot.
Start(): c gi trc khi khung hnh u tin c cp nht.
Update(): Gi mi khung hnh.
LateUpdate(): c gi mi khung hnh sau khi hm Update().
OnBecameVisible(): c gi khi i tng tr chi hin ti tr nn hin th thng qua bt k camera no.
OnBecameInvisible(): c gi khi i tng tr chi hin ti khng cn hin th qua bt k camera no.
OnDrawGizmos(): c gi  v gizmos trong ca s scene.
OnGUI(): c gi nhiu ln cho cc s kin GUI.
OnApplicationPause(): c gi khi tr chi tm dng thng qua Unity editor.
OnDisable(): Thc thi khi i tng tr chi b tt.
OnDestroy(): c kch hot khi i tng tr chi b hy.

C mt hm vng i c gi l FixedUpdate, c kch hot mt s ln c nh mi giy. Bn c th cu hnh tn sut trong Edit Project Settings Time Fixed Timestep.

S kin vt l

Nu bn thm mt thnh phn collider vo mt i tng trong game, bn c th pht hin s kin va chm t cc thnh phn ca bn bng cch nh ngha mt b phn c th ca cc phng thc. Cc phng thc sau ch c gi khi bn c mt Collider hoc Rigid Body component trn c hai i tng trong game.

  • OnCollisionEnter - Phng thc ny c gi mt ln khi mt i tng khc va chm vi i tng game hin ti.
  • OnCollisionStay - Phng thc ny c gi trong mi khung hnh khi mt i tng khc va chm vi i tng game hin ti.
  • OnCollisionExit - Phng thc ny c gi mt ln khi mt i tng ri khi vng va chm ca i tng hin ti.
private void OnCollisionEnter(Collision hit) {
  Debug.Log($"{gameObject.name} hits {hit.gameObject.name}");
}
private void OnCollisionStay(Collision hit) {
  Debug.Log($"{gameObject.name} is hitting {hit.gameObject.name}");
}
private void OnCollisionExit(Collision hit) {
  Debug.Log($"{gameObject.name} stopped hitting {hit.gameObject.name}");
}

Cc hm sau ch c gi khi ty chn On Trigger c bt t thnh phn collider tng ng.
-OnTriggerEnter - Hm ny c gi khi mt i tng khc va chm vi i tng game hin ti.
-OnTriggerStay - Hm ny c gi trong mi khung hnh khi mt i tng khc va chm vi i tng game hin ti.
-OnTriggerExit - Hm ny c gi mt ln khi mt i tng ri khi vng va chm ca i tng hin ti.

Serializing

Unity c kh nng serialize cc bin, chuyn i d liu thnh mt nh dng c th chnh sa t trnh son tho Unity. Qu trnh serialization ph thuc vo cc thuc tnh hoc cc trnh iu khin truy cp c p dng cho bin.

Nu mt bin l public, n s c t ng serialize:

public int age = 10;

Nu bn khng mun bin public c serialize, bn c th s dng thuc tnh NonSerialized t namespace System nh sau:

[NonSerialized] public int age = 10;

Bin private khng c serialize mc nh. Tuy nhin, nu bn mun chng c serialize, Unity c mt thuc tnh gi l SerializeField trong namespace Unity. Bn c th p dng n nh sau:

[SerializeField] private int age = 10;

Khi to

Cc i tng game mi c th c chn vo cnh mt cch t ng thng qua vic gi hm sau. Hm ny c ba i s.

  • i tng Game - The Game Object
  • (Optional) V tr Ton cu - Global Position
  • (Optional) Quay - Rotation
Instantiate(someGameObject);
Instantiate(someGameObject, new Vector3(0, 0, 10));
Instantiate(someGameObject, new Vector3(0, 0, 10), Quaternion.identity);

Tm i Tng Trong Game

Lp GameObject c mt s phng thc tm kim i tng game trong cu trc th mc.

  • Find() - Tm kim mt i tng game da trn tn ca n.
  • FindGameObjectsWithTag() - Tm nhiu i tng game c lu tr trong mt mng v tr v. S dng tag ca i tng game tm chng.
  • FindWithTag() - Tm mt i tng game da trn tag ca n.
GameObject myObj = GameObject.Find("NAME IN HIERARCHY");
GameObject myObj = GameObject.FindGameObjectsWithTag("TAG");
GameObject myObj = GameObject.FindWithTag("TAG");

Cc Thnh Phn (Components)

Cc i tng game t nhin khng c nhiu chc nng. Chng ta cn thm cc thnh phn c cc chc nng c th. Nu bn ang s dng cc thnh phn ty chnh v cn chn la cc thnh phn khc, Unity cung cp cch ly chng. Phng php u tin l s dng hm GetComponent(), thng bng cch ch nh tn lp thnh phn.

AudioSource audioSource = GetComponent<AudioSource>();

Hoc bn c th s dng t kha typeof v xc nhn gi tr nh sau:

AudioSource audioSource = GetComponent(typeof(AudioSource)) as AudioSource;

Cui cng, bn c th truyn vo tn ca thnh phn di dng chui nh sau:

AudioSource audioSource = GetComponent("AudioSource") as AudioSource;

Hng S (Constants)

Trong Unity, bn c th s dng hng s nh ngha cc gi tr gi nguyn trong ton b kch bn ca bn. Di y l cch bn c th s dng hng s trong Unity:

  1. Khai Bo Mt Hng S:
    S dng t kho const khai bo mt hng s. Hng s phi c khi to gi tr ti thi im khai bo v khng th thay i sau . Di y l mt v d:
public class ExampleScript : MonoBehaviour
{
    // Define a constant for gravity
    private const float Gravity = 9.8f;

    void Start()
    {
        // Use the constant in your code
        float fallSpeed = Gravity * Time.deltaTime;
    }
}

  1. Li ch ca Hng S:
    Hng s mang li nhng li ch quan trng khi nh ngha cc gi tr c s dng nhiu ni trong m ca bn v khng nn c sa i trong thi gian chy. Chng gip m ca bn tr nn d c v d bo tr hn.

  2. S Dng Trong Unity:
    Bn c th s dng hng s trong nhiu kch bn Unity khc nhau, chng hn nh cc kch bn MonoBehaviour c nh km vo GameObjects hoc cc kch bn tin ch. Hng s thng c s dng cho cc gi tr nh tc mc nh, kch thc c nh hoc cc tham s khng thay i khc.

  3. Phm Vi ca Hng S:
    Hng s c phm vi gii hn trong lp hoc phng thc m chng c khai bo. Nu bn cn hng s c th truy cp t nhiu kch bn, hy xem xt vic s dng mt lp public static.

Di y l mt v d n gin:

public static class GameConstants
{
    public const int MaxScore = 100;
}

public class ScoreManager : MonoBehaviour
{
    void Update()
    {
        // Access the constant from another script
        int currentScore = CalculateScore();
        if (currentScore >= GameConstants.MaxScore)
        {
            Debug.Log("You reached the maximum score!");
        }
    }

    int CalculateScore()
    {
        // Your score calculation logic here
        return 50;
    }
}

Vectors

Vectors quyt nh v tr ca cc i tng trong game v h tr trong cc php tnh v khong cch v chuyn ng. Unity c hai lp vector: Vector2 cho cc tr chi 2D vi ta (x, y), v Vector3 cho cc tr chi 3D vi ta (x, y, z). Trong cc lp ny, X i din cho Bn tri/Phi, Y biu th Ln/Xung, v Z cho Tin ln/Li li. Unity cung cp cc hng s c th truy cp t lp vector tng ng.

Vector 3

Vector3 V = new Vector3(0f; 0f; 0f);

Vector3.right /* (1, 0, 0)  */
Vector3.left /* (-1, 0, 0)  */
Vector3.up /* (0, 1, 0)  */
Vector3.down /* (0, -1, 0)  */
Vector3.forward /* (0, 0, 1)  */
Vector3.back /* (0, 0, -1) */
Vector3.zero /* (0, 0, 0)  */
Vector3.one /* (1, 1, 1)  */

Vector 2

Vector2.right /* (1, 0)  */
Vector2.left  /* (-1, 0) */
Vector2.up    /* (0, 1)  */
Vector2.down  /* (0, -1) */
Vector2.zero  /* (0, 0)  */
Vector2.one   /* (1, 1)  */

Nu bn ch quan tm n hng ca mt vector c th, bn c th truy cp thuc tnh normalized ca n nh sau:

myVector.normalized

Khong cch gia hai vector c th c tnh bng phng thc Vector3.Distance(), tr v mt gi tr kiu float:

float distance = Vector3.Distance(vectorOne, vectorTwo);

Quaternion

Quaternions iu khin xoay ca game object. Bn c th kim tra hoc thay i xoay ca i tng bng cch s dng transform.rotation. Hm Quaternion.LookRotation() ca Unity to ra mt xoay da trn mt v tr mc tiu.

Quaternion.LookRotation(gameObjectPosition);

Mt xoay 30 quanh trc y:

Quaternion rotation = Quaternion.Euler(0, 30, 0);

Gc Euler

Gc Euler l gc nh 90, 180, 45, 30 . Tuy nhin, quaternion khc bit so vi gc Euler ch n biu din mt im trn Mt cu n v (bn knh l 1 n v).

V d s dng Quaternion biu din 30 xoay quanh trc X v 10 xoay quanh trc Y:

Quaternion rotation = Quaternion.Euler(30, 10, 0);

S dng mt Vector biu din cng l mt cch khc:

Vector3 EulerRotation = new Vector3(30, 10, 0);
Quaternion rotation = Quaternion.Euler(EulerRotation);

Chuyn i gc Quaternion ca mt transform thnh gc Euler

Quaternion quaternionAngles = transform.rotation;
Vector3 eulerAngles = quaternionAngles.eulerAngles;

Gi T L Mn Hnh 

using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class ScaleBackground : MonoBehaviour
{
    private SpriteRenderer background;
    private Camera cam;

    void Start()
    {
        background = GetComponent<SpriteRenderer>();
        cam = Camera.main;

        // Ly kch thc ca camera
        float cameraHeight = 2f * cam.orthographicSize;
        float cameraWidth = cameraHeight * cam.aspect;

        // Ly kch thc ca nn hin ti
        float bgHeight = background.sprite.bounds.size.y;
        float bgWidth = background.sprite.bounds.size.x;

        // Tnh t l t l
        float scaleX = cameraWidth / bgWidth;
        float scaleY = cameraHeight / bgHeight;

        // p dng t l cho nn
        transform.localScale = new Vector3(scaleX, scaleY, 1f);
    }
  }
}

Singleton

// nh ngha lp singleton
public class SingletonClass: MonoBehaviour {
    private static SomeClass instance;

    public static SomeClass Instance { get { return instance; } }

    private void Awake() {
        if (instance != null && instance != this) {
            Destroy(this.gameObject);
        } else {
            instance = this;
        }
    }
}

// S dng n trong mt lp khc
public class AnotherClass: MonoBehaviour {
    public Singleton instance;

    private void Awake() {
       instance = Singleton.Instance;
    }
}
